# pousada-dashboard

PWA simples para controle diário de apartamentos e caixa da pousada.

Arquivos incluídos:
- index.html (PWA + interface)
- manifest.json
- sw.js (service worker)
- icons-instructions.txt (links para ícones placeholder)

Como publicar (GitHub Pages):
1. Vá em Settings > Pages
2. Escolha Branch: main e Folder: / (root)
3. Clique Save
4. Acesse: https://akilessimiao.github.io/pousada-dashboard

Notas:
- Atualizei o número do WhatsApp no arquivo index.html para o número fornecido (5584987192094) sem o sinal +. Se preferir outro número, edite a variável `numero` no script.
- Os ícones não foram incluidos como binários; veja icons-instructions.txt para links de placeholder e adicione arquivos `icon-192.png`, `icon-512.png` e `favicon.ico` na raiz do repositório.
